Sitespeed.io 2.2.1 release notes

This release contains a couple of bug fixes and an upgrade of BrowserTime that makes it possible to test using Firefox 25:

  • New BrowserTime version (0.2) that fixes the Firefox 25 bug & now have a 60 seconds wait for load before breaking (old version had 30 seconds).
  • Logging all PhantomJS errors to own PhantomJS error log (since there is a defect in QT using Mavericks).
  • Bug fix: Unable to crawl websites with GA cookie #298
  • Bug fix: sitespeed-sites.io used the sitespeed.io script with sh instead of bash

And there is one new small feature: You can now supply a test name that will be showed on all test result pages (using the parameter n). See an example here.

See the changelog for changes done in the past.
