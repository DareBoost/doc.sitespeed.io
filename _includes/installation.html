Install via Homebrew on Mac

1. Install sitespeed.io

$ brew install sitespeedio/sitespeedio/sitespeed.io

2. Run

$ sitespeed.io -h

Install other *nix

1. Install PhantomJS (1.9)

2. Install/use Java 1.6 (or higher)

3. Clone the project from Github 

$ git clone https://github.com/sitespeedio/sitespeed.io.git

or download and unpack the zip.

4. Run

$ bin/sitespeed.io -h
