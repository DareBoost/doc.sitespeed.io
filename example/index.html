Examples

Here's a couple of examples of how the result looks like for an analyzis made by sitespeed.

Testing one site:

./sitespeed.io -u http://www.cybercom.com  -c true -a iphone -z "-b chrome" -k true

The result

Testing multiple sites:

./sitespeed-sites.io  -c true -a iphone -z "-b chrome" -j 10

Testing Alexa top sites.
